# Expense-Tracker
# 💰 Expense Tracker (C++)

## 📌 Description

This is a simple C++ based Expense Tracker that allows users to manage their balance and record daily expenses. It is built using Object-Oriented Programming (OOP) concepts and dynamic memory allocation.

---

## 🚀 Features

* User Login System
* Deposit & Spend Money
* Add Expenses with Description
* View Balance & Expense History
* Expense Summary

---

## 🧠 Concepts Used

* Classes & Objects
* Constructors & Destructor
* Operator Overloading
* Friend Function
* Dynamic Memory Allocation
